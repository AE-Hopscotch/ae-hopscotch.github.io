GitHub Test 2

After getting a newer device and being unable to access my original test project from it, I decided to create the second rendition of that project. Here is where I will test out new webplayer mods such as global variables, getting user info, and other actions that are yet to exist in Hopscotch. This project is linked directly to a draft, meaning that every time I update the draft with new features, this project page will automatically be updated to show these changes. In this project, I've shown so far the use of a link action, retriving user information from Hopscotch's servers, and setting global variables that work across multiple projects by the same user.

Shift-Click Minesweeper

This is the first game I added keyboard support to. Instead of having to click the flag button every time when you are on a computer, youu can now shift+click on a tile to flag it. I have other projects in progress with this mod, but this was the first game I supported with it. Unfortauntely, it is not a published version that you can find on my profile, but it does work just like v1.1, except that this one has keyboard support whereas v1.1 does not. Like the original version and v1.1, the objective of the game itself is to clear the entire board by tapping on all safe tiles, where each safe tile indicates the number of mines around it.
